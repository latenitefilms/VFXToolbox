# VFX Toolbox

**VFX Toolbox** is for visual effects editors and post-production supervisors working with Final Cut Pro.

![](/static/vfx-toolbox-01.png)

It provides seven integrated tools for pulling VFX plates from timelines, naming shots, generating title overlays, importing VFX deliveries, extracting shot lists, comparing versions, and exporting metadata codebooks - all driven by `FCPXML`.
